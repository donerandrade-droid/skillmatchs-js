# SkillMatch JS

## O que é esse projeto?
É um simulador que compara as habilidades de um candidato com os requisitos de vagas de front-end júnior.

Ele mostra:
- se o candidato tem as habilidades que a vaga pede
- quais habilidades estão faltando
- qual vaga tem mais compatibilidade
- uma recomendação do que estudar

## Como rodar
No terminal: node skillmatch.js

## no navegador:
1. Abrir o Chrome
2. Apertar F12 para abrir o console
3. Colar o código do arquivo skillmatch.js
4. Apertar Enter

- usar if/else para classificar resultados
